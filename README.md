# WealthWise - Smart Personal Finance Manager

An AI-powered financial dashboard that connects to bank accounts, analyzes spending patterns, provides investment insights, and helps users achieve financial goals.

## Features

- **Smart Categorization**: AI automatically categorizes your transactions.
- **Budget Tracking**: Set limits and get notified before you overspend.
- **Goal Planning**: Visualize your progress towards financial freedom.
- **Investment Insights**: Get personalized/AI-driven investment advice.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS, Shadcn UI
- **Backend**: Supabase
- **Icons**: Lucide React
- **Charts**: Recharts

## Getting Started

1. Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials:
   ```bash
   cp .env.local.example .env.local
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
